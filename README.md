# Excel Skills for Business: Intermediate II

This repository contains my practice work from the *Excel Skills for Business: Intermediate II* course (part of the Excel Skills for Business Specialization by Macquarie University on Coursera).

## Purpose

This repository is created to:

- Strengthen intermediate-to-advanced Excel skills through structured hands-on practice  
- Document weekly learning progress  
- Maintain consistent GitHub activity  
- Build stronger spreadsheet management and data validation skills  

This repository serves as a structured learning archive rather than a portfolio project.

## Contents

### Week 1 – Data Validation and Conditional Formatting

- Applying data validation to enforce data integrity  
- Creating and managing drop-down lists  
- Using formulas within data validation  
- Preventing duplicate or invalid entries using custom rules  
- Working with input messages and error alerts  
- Introduction to conditional formatting  
- Applying basic and advanced conditional formatting rules  

## Key Skills Practiced

- Data validation and input control  
- Custom formula-based validation  
- Error handling in spreadsheets  
- Data quality management  
- Conditional formatting for data insights  

## Tools Used

- Microsoft Excel
